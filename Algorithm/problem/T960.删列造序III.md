## 代码

```java
public int minDeletionSize(String[] strs) {  
    int res = 0;  
    int n = strs[0].length();  
    // f[j] 以第 j 列结尾的最长递增子序列长度
    int[] f = new int[n];  
    for(int j = 0;j < n;++j) {  
        for(int i = 0;i < j;++i) {  
	        // 加上f[i] > f[j] 可以减少check的运行次数
            if(f[i] > f[j] && check(strs, i, j)) {  
	            // f[j] = Math.max(f[j], f[i])
                f[j] = f[i];  
            }  
        }  
        f[j]++;  
        res = Math.max(res, f[j]);  
    }  
    return n - res;  
}  
private boolean check(String[] strs, int i, int j) {  
    for(String s: strs) {  
        if(s.charAt(i) > s.charAt(j)) {  
            return false;  
        }  
    }  
    return true;  
}
```
